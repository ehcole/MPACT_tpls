SNESGetSolution

 Returns the vector where the approximate solution is stored.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetSolution(SNES snes,Vec *x)
Not Collective, but Vec is parallel if SNES is parallel

Input Parameter

snes - the SNES context

Output Parameter

x - the solution

Keywords

SNES, nonlinear, get, solution

See Also

SNESGetSolutionUpdate(), SNESGetFunction()
