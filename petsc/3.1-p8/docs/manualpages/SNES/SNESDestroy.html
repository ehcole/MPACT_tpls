SNESDestroy

 Destroys the nonlinear solver context that was created with SNESCreate().

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESDestroy(SNES snes)

Input Parameter

snes -the SNES context
