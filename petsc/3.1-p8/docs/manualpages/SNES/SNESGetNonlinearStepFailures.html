SNESGetNonlinearStepFailures

 Gets the number of unsuccessful steps attempted by the nonlinear solver.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetNonlinearStepFailures(SNES snes,PetscInt* nfails)
Not Collective

Input Parameter

snes -SNES context

Output Parameter

nfails -number of unsuccessful steps attempted

Notes

This counter is reset to zero for each successive call to SNESSolve().

Keywords

SNES, nonlinear, get, number, unsuccessful, steps
