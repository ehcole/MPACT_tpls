SNESGetLinearSolveIterations

 Gets the total number of linear iterations used by the nonlinear solver.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetLinearSolveIterations(SNES snes,PetscInt* lits)
Not Collective

Input Parameter

snes -SNES context

Output Parameter

lits -number of linear iterations

Notes

This counter is reset to zero for each successive call to SNESSolve().

Keywords

SNES, nonlinear, get, number, linear, iterations

See Also
