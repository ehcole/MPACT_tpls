SNESGetMaxNonlinearStepFailures

 Gets the maximum number of unsuccessful steps attempted by the nonlinear solver before it gives up.

Synopsis

#include "petscsnes.h"  
PetscErrorCode PETSCSNES_DLLEXPORT SNESGetMaxNonlinearStepFailures(SNES snes, PetscInt *maxFails)
Not Collective

Input Parameter

snes -SNES context

Output Parameter

maxFails -maximum of unsuccessful steps

Keywords

SNES, nonlinear, get, maximum, unsuccessful, steps
