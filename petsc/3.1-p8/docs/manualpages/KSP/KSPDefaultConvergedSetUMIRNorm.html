KSPDefaultConvergedSetUMIRNorm

 makes the default convergence test use min(|| B*(b - A*(initial guess))||,|| B*b ||) In the case of right preconditioner or if KSPSetNormType(ksp,KSP_NORM_UNPRECONDIITONED) is used there is no B in the above formula. UMIRNorm is short for Use Minimum Initial Residual Norm.

Synopsis

#include "petscksp.h" 
PetscErrorCode PETSCKSP_DLLEXPORT KSPDefaultConvergedSetUMIRNorm(KSP ksp)
Collective on KSP

Input Parameters

ksp -iterative context

Options Database

-ksp_converged_use_min_initial_residual_norm - Use KSPSetTolerances() to alter the defaults for rtol, abstol, dtol.

The precise values of reason are macros such as KSP_CONVERGED_RTOL, which are defined in petscksp.h.

Keywords

KSP, default, convergence, residual
