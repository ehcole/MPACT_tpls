PetscDrawClear

 Clears graphical output.

Synopsis

#include "petscdraw.h" 
PetscErrorCode  PetscDrawClear(PetscDraw draw)
Not collective (Use PetscDrawSynchronizedClear() for collective)

Input Parameter

draw -the drawing context
