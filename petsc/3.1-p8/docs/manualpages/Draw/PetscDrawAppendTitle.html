PetscDrawAppendTitle

 Appends to the title of a PetscDraw context.

Synopsis

#include "petscdraw.h" 
PetscErrorCode PETSC_DLLEXPORT PetscDrawAppendTitle(PetscDraw draw,const char title[])
Not collective (any processor or all can call this)

Input Parameters

draw - the graphics context
title - the title

Note

A copy of the string is made, so you may destroy the title string after calling this routine.

See Also

PetscDrawSetTitle(), PetscDrawGetTitle()
