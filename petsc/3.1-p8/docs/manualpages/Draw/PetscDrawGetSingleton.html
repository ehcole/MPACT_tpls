PetscDrawGetSingleton

 Gain access to a PetscDraw object as if it were owned by the one process.

Synopsis

#include "petscdraw.h" 
PetscErrorCode PETSC_DLLEXPORT PetscDrawGetSingleton(PetscDraw draw,PetscDraw *sdraw)
Collective on PetscDraw

Input Parameter

draw -the original window

Output Parameter

sdraw -the singleton window

See Also

PetscDrawRestoreSingleton(), PetscViewerGetSingleton(), PetscViewerRestoreSingleton()
