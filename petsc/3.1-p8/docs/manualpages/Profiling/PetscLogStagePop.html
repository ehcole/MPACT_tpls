PetscLogStagePop

 This function pops a stage from the stack.

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscLogStagePop(void)
Not Collective

Usage

If the option -log_sumary is used to run the program containing the following code, then 2 sets of summary data will be printed during PetscFinalize(). 
      PetscInitialize(int *argc,char ***args,0,0);
      [stage 0 of code]   
      PetscLogStagePush(1);
      [stage 1 of code]
      PetscLogStagePop();
      PetscBarrier(...);
      [more stage 0 of code]   
      PetscFinalize();

Notes

Use PetscLogStageRegister() to register a stage.

Keywords

log, pop, stage

See Also

PetscLogStagePush(), PetscLogStageRegister(), PetscBarrier()
