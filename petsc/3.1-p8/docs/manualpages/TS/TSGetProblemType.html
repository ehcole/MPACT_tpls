TSGetProblemType

 Gets the type of problem to be solved.

Synopsis

#include "petscts.h"  
PetscErrorCode PETSCTS_DLLEXPORT TSGetProblemType(TS ts, TSProblemType *type)
Not collective

Input Parameter

ts -The TS

Output Parameter

type -One of TS_LINEAR, TS_NONLINEAR where these types refer to problems of the forms 
         U_t = A U    
         U_t = A(t) U 
         U_t = F(t,U)

Keywords

TS, problem type

See Also

TSSetUp(), TSProblemType, TS
