TSSetTimeStep

 Allows one to reset the timestep at any time, useful for simple pseudo-timestepping codes.

Synopsis

#include "petscts.h"  
PetscErrorCode PETSCTS_DLLEXPORT TSSetTimeStep(TS ts,PetscReal time_step)
Collective on TS

Input Parameters

ts - the TS context obtained from TSCreate()
time_step - the size of the timestep

See Also

TSSetInitialTimeStep(), TSGetTimeStep()

Keywords

TS, set, timestep
