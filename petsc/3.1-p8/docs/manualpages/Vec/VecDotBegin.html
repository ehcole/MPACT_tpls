VecDotBegin

 Starts a split phase dot product computation.

Synopsis

#include "petscvec.h"   
PetscErrorCode PETSCVEC_DLLEXPORT VecDotBegin(Vec x,Vec y,PetscScalar *result)

Input Parameters

x - the first vector
y - the second vector
result - where the result will go (can be PETSC_NULL)

C++ variants

  VecDotBegin(Vec x,Vec y)->VecDotBegin(x,y,PETSC_NULL)

Notes

Each call to VecDotBegin() should be paired with a call to VecDotEnd().

seealso: VecDotEnd(), VecNormBegin(), VecNormEnd(), VecNorm(), VecDot(), VecMDot(), VecTDotBegin(), VecTDotEnd()
