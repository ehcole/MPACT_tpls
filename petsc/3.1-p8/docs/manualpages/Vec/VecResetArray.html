VecResetArray

 Resets a vector to use its default memory. Call this after the use of VecPlaceArray().

Synopsis

#include "petscvec.h" 
PetscErrorCode PETSCVEC_DLLEXPORT VecResetArray(Vec vec)
Not Collective

Input Parameters

vec - the vector

See Also
