PetscLayoutGetSize

 Gets the global size for a PetscLayout object.

Synopsis

#include "petscvec.h"   
#undef __FUNCT__  
#define __FUNCT__ "PetscLayoutGetSize"
PetscErrorCode PETSCVEC_DLLEXPORT PetscLayoutGetSize(PetscLayout map,PetscInt *n)
Not Collective

Input Parameters

map -pointer to the map

Output Parameters

n -the global size

Notes

Call this after the call to PetscLayoutSetUp()

Fortran Notes

Not available from Fortran
