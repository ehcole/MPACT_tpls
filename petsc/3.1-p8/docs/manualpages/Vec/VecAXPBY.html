VecAXPBY

 Computes y = alpha x + beta y.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecAXPBY(Vec y,PetscScalar alpha,PetscScalar beta,Vec x)
Collective on Vec

Input Parameters

alpha,beta - the scalars
x, y - the vectors

Output Parameter

y -output vector

Notes: x and y must be different vectors

See Also

VecAYPX(), VecMAXPY(), VecWAXPY(), VecAXPY()
