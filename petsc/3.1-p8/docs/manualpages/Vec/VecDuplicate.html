VecDuplicate

 Creates a new vector of the same type as an existing vector.

Synopsis

#include "petscvec.h" 
PetscErrorCode PETSCVEC_DLLEXPORT VecDuplicate(Vec v,Vec *newv)
Collective on Vec

Input Parameters

v -a vector to mimic

Output Parameter

newv -location to put new vector

Notes

VecDuplicate() does not copy the vector, but rather allocates storage for the new vector. Use VecCopy() to copy a vector.

Use VecDestroy() to free the space. Use VecDuplicateVecs() to get several vectors.

See Also

VecDestroy(), VecDuplicateVecs(), VecCreate(), VecCopy()
