VecAYPX

 Computes y = x + alpha y.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecAYPX(Vec y,PetscScalar alpha,Vec x)
Collective on Vec

Input Parameters

alpha - the scalar
x, y - the vectors

Output Parameter

y -output vector

Notes: x and y must be different vectors

See Also

VecAXPY(), VecWAXPY()
