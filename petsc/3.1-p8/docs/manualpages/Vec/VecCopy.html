VecCopy

 Copies a vector. y <- x

Synopsis

#include "petscvec.h" 
PetscErrorCode PETSCVEC_DLLEXPORT VecCopy(Vec x,Vec y)
Collective on Vec

Input Parameter

x -the vector

Output Parameter

y -the copy

Notes

For default parallel PETSc vectors, both x and y must be distributed in the same manner; local copies are done.

See Also

VecDuplicate()
