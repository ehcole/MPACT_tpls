MatGetRowMinAbs

 Gets the minimum value (in absolute value) of each row of the matrix

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatGetRowMinAbs(Mat mat,Vec v,PetscInt idx[])
Collective on Mat and Vec

Input Parameters

mat -the matrix

Output Parameter

v - the vector for storing the minimums
idx - the indices of the column found for each row (optional)

Notes: if a row is completely empty or has only 0.0 values then the idx[] value for that row is 0 (the first column).

This code is only implemented for a couple of matrix formats.
