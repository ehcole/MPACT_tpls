MatMFFDSetFunctioniBase

 Sets the base vector for a single component function evaluation

Synopsis

#include "petscmat.h"   
PetscErrorCode PETSCMAT_DLLEXPORT MatMFFDSetFunctioniBase(Mat mat,PetscErrorCode (*func)(void*,Vec))
Collective on Mat

Input Parameters

mat - the matrix free matrix created via MatCreateSNESMF()
func - the function to use

Notes

If you use this you MUST call MatAssemblyBegin()/MatAssemblyEnd() on the matrix free matrix inside your compute Jacobian routine

Keywords

SNES, matrix-free, function
