MatGetDiagonal

 Gets the diagonal of a matrix.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatGetDiagonal(Mat mat,Vec v)
Collective on Mat and Vec

Input Parameters

mat - the matrix
v - the vector for storing the diagonal

Output Parameter

v -the diagonal of the matrix
