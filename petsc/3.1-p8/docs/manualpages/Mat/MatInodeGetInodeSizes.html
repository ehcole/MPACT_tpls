MatInodeGetInodeSizes

 Returns the inode information of the Inode matrix.

Synopsis

PetscErrorCode PETSCMAT_DLLEXPORT MatInodeGetInodeSizes(Mat A,PetscInt *node_count,PetscInt *sizes[],PetscInt *limit)
Collective on Mat

Input Parameter

A -the Inode matrix or matrix derived from the Inode class -- e.g., SeqAIJ

Output Parameter

node_count - no of inodes present in the matrix.
sizes - an array of size node_count,with sizes of each inode.
limit - the max size used to generate the inodes.

Notes: This routine returns some internal storage information of the matrix, it is intended to be used by advanced users. It should be called after the matrix is assembled. The contents of the sizes[] array should not be changed. PETSC_NULL may be passed for information not requested.

Keywords

matrix, seqaij, get, inode
