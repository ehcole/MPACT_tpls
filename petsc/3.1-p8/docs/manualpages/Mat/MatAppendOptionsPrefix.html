MatAppendOptionsPrefix

 Appends to the prefix used for searching for all Mat options in the database.

Synopsis

#include "petscmat.h" 
PetscErrorCode PETSCMAT_DLLEXPORT MatAppendOptionsPrefix(Mat A,const char prefix[])
Collective on Mat

Input Parameters

A - the Mat context
prefix - the prefix to prepend to all option names

Notes

A hyphen (-) must NOT be given at the beginning of the prefix name. The first character of all runtime options is AUTOMATICALLY the hyphen.

Keywords

Mat, append, options, prefix, database
