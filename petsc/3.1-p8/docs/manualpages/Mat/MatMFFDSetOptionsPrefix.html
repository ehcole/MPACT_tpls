MatMFFDSetOptionsPrefix

 Sets the prefix used for searching for all MatMFFD options in the database.

Synopsis

#include "petscmat.h"   
PetscErrorCode PETSCMAT_DLLEXPORT MatMFFDSetOptionsPrefix(Mat mat,const char prefix[])
Collective on Mat

Input Parameter

A - the Mat context
prefix - the prefix to prepend to all option names

Notes

A hyphen (-) must NOT be given at the beginning of the prefix name. The first character of all runtime options is AUTOMATICALLY the hyphen.

Keywords

SNES, matrix-free, parameters

See Also

MatMFFDSetFromOptions(), MatCreateSNESMF()
