MatMultTransposeConstrained

 The inner multiplication routine for a constrained matrix P^T A^T P.

Synopsis

#include "petscmat.h" 
PetscErrorCode PETSCMAT_DLLEXPORT MatMultTransposeConstrained(Mat mat,Vec x,Vec y)
Collective on Mat and Vec

Input Parameters

mat - the matrix
x - the vector to be multilplied

Output Parameters

y -the result

Notes

The vectors x and y cannot be the same. I.e., one cannot call MatMult(A,y,y).

Keywords

matrix, multiply, matrix-vector product, constraint

See Also

MatMult(), MatMultTranspose(), MatMultAdd(), MatMultTransposeAdd()
