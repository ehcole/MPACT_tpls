MatGetBlockSize

 Returns the matrix block size; useful especially for the block row and block diagonal formats.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatGetBlockSize(Mat mat,PetscInt *bs)
Not Collective

Input Parameter

mat -the matrix

Output Parameter

bs -block size

C++ variants

PetscInt MatGetBlockSize(Mat mat)->MatGetBlockSize(mat,&a); return a;

Notes

Block row formats are MATSEQBAIJ, MATMPIBAIJ, MATSEQSBAIJ, MATMPISBAIJ

See Also

MatCreateSeqBAIJ(), MatCreateMPIBAIJ()
