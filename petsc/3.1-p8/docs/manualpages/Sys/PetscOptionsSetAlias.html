PetscOptionsSetAlias

 Makes a key and alias for another key

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscOptionsSetAlias(const char inewname[],const char ioldname[])
Not Collective, but setting values on certain processors could cause problems for parallel objects looking for options.

Input Parameters

inewname - the alias
ioldname - the name that alias will refer to
