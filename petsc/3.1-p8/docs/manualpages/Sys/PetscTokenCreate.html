PetscTokenCreate

 Creates a PetscToken used to find tokens in a string

Synopsis

#include "petscsys.h"   
PetscErrorCode PETSC_DLLEXPORT PetscTokenCreate(const char a[],const char b,PetscToken *t)
Not Collective

Input Parameters

string - the string to look in
token - the character to look for

Output Parameter

a -pointer to token

Notes

This version is different from the system version in that it allows you to pass a read-only string into the function.

See Also

PetscTokenFind(), PetscTokenDestroy()
