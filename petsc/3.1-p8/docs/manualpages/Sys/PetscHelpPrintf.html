PetscHelpPrintf

 Prints help messages.

Synopsis

PetscErrorCode (*PetscHelpPrintf)(const char format[],...);
Not Collective

Input Parameters

format -the usual printf() format string

Fortran Note

This routine is not supported in Fortran.
