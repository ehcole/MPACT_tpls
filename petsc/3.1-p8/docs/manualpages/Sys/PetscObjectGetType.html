PetscObjectGetType

 Gets the object type of any PetscObject.

Synopsis

#include "petscsys.h"    
PetscErrorCode  PetscObjectGetType(PetscObject obj, const char *type[])
Not Collective

Input Parameter

obj -any PETSc object, for example a Vec, Mat or KSP. Thus must be cast with a (PetscObject), for example, PetscObjectGetType((PetscObject)mat,&type);

Output Parameter

type -the object type

Level:advanced
