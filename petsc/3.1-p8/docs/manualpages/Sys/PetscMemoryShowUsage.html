PetscMemoryShowUsage

 Shows the amount of memory currently being used in a communicator.

Synopsis

#include "petscsys.h" 
PetscErrorCode  PetscMemoryShowUsage(PetscViewer viewer,const char message[])
Collective on PetscViewer

Input Parameter

viewer - the viewer that defines the communicator
message - string printed before values

See Also

PetscMallocDump(), PetscMemoryGetCurrentUsage()
