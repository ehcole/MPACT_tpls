PetscStrallocpy

 Allocates space to hold a copy of a string then copies the string

Synopsis

#include "petscsys.h"   
PetscErrorCode  PetscStrallocpy(const char s[],char *t[])
Not Collective

Input Parameters

s -pointer to string

Output Parameter

t -the copied string

Note

Null string returns a new null string
