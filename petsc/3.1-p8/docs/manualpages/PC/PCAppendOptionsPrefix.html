PCAppendOptionsPrefix

 Appends to the prefix used for searching for all PC options in the database.

Synopsis

#include "petscksp.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCAppendOptionsPrefix(PC pc,const char prefix[])
Collective on PC

Input Parameters

pc - the preconditioner context
prefix - the prefix string to prepend to all PC option requests

Notes

A hyphen (-) must NOT be given at the beginning of the prefix name. The first character of all runtime options is AUTOMATICALLY the hyphen.

Keywords

PC, append, options, prefix, database

See Also

PCSetOptionsPrefix(), PCGetOptionsPrefix()
