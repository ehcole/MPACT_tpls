PCMGGetSmootherDown

 Gets the KSP context to be used as smoother before coarse grid correction (pre-smoother).

Synopsis

#include "petscksp.h" 
#include "petscmg.h"   
PetscErrorCode PETSCKSP_DLLEXPORT PCMGGetSmootherDown(PC pc,PetscInt l,KSP *ksp)
Not Collective, KSP returned is parallel if PC is

Input Parameters

pc - the multigrid context
l - the level (0 is coarsest) to supply

Ouput Parameters

ksp -the smoother

Keywords

MG, multigrid, get, smoother, down, pre-smoother, level
