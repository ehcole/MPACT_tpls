PCMGSetR

 Sets the vector space to be used to store the residual on a particular level.

Synopsis

#include "petscksp.h" 
#include "petscmg.h"   
PetscErrorCode PETSCKSP_DLLEXPORT PCMGSetR(PC pc,PetscInt l,Vec c)
Collective on PC and Vec

Input Parameters

pc - the multigrid context
l - the level (0 is coarsest) this is to be used for
c - the space

Notes: If this is not provided PETSc will automatically generate one.

You do not need to keep a reference to this vector if you do not need it PCDestroy() will properly free it.

Keywords

MG, multigrid, set, residual, level

Level:advanced
