PCShellSetContext

 sets the context for a shell PC

Synopsis

#include "petscpc.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCShellSetContext(PC pc,void *ctx)
Collective on PC

Input Parameters

pc - the shell PC
ctx - the context

Fortran Notes: The context can only be an integer or a PetscObject unfortunately it cannot be a Fortran array or derived type.

See Also

PCCreateShell(), PCShellGetContext()
