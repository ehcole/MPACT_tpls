PCFactorSetReuseFill

 When matrices with same different nonzero structure are factored, this causes later ones to use the fill ratio computed in the initial factorization.

Synopsis

#include "petscpc.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCFactorSetReuseFill(PC pc,PetscTruth flag)
Collective on PC

Input Parameters

pc - the preconditioner context
flag - PETSC_TRUE to reuse else PETSC_FALSE

Options Database Key

-pc_factor_reuse_fill -Activates PCFactorSetReuseFill()

Keywords

PC, levels, reordering, factorization, incomplete, Cholesky
