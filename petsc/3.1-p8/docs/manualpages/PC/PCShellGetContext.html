PCShellGetContext

 Returns the user-provided context associated with a shell PC

Synopsis

#include "petscpc.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCShellGetContext(PC pc,void **ctx)
Not Collective

Input Parameter

pc -should have been created with PCCreateShell()

Output Parameter

ctx -the user provided context

Notes

This routine is intended for use within various shell routines

Keywords

PC, shell, get, context

See Also

PCCreateShell(), PCShellSetContext()
