PCRedundantSetNumber

 Sets the number of redundant preconditioner contexts.

Synopsis

#include "petscpc.h" 
PetscErrorCode PETSCKSP_DLLEXPORT PCRedundantSetNumber(PC pc,PetscInt nredundant)
Collective on PC

Input Parameters

pc - the preconditioner context
nredundant - number of redundant preconditioner contexts; for example if you are using 64 MPI processes and use an nredundant of 4 there will be 4 parallel solves each on 16 = 64/4 processes.

Keywords

PC, redundant solve

Level:advanced
