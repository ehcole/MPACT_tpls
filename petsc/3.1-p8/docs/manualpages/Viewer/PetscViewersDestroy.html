PetscViewersDestroy

 Destroys a set of PetscViewers created with PetscViewersCreate().

Synopsis

PetscErrorCode PETSC_DLLEXPORT PetscViewersDestroy(PetscViewers v)
Collective on PetscViewers

Input Parameters

v -the PetscViewers to be destroyed.
