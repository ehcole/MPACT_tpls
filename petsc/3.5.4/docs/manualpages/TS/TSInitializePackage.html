TSInitializePackage

 This function initializes everything in the TS package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate() when using static libraries.

Synopsis

PetscErrorCode  TSInitializePackage(void)

Keywords

TS, initialize, package
