TSPreStep

 Runs the user-defined pre-step function.

Synopsis

#include "petscts.h"  
PetscErrorCode  TSPreStep(TS ts)
Collective on TS

Input Parameters

ts -The TS context obtained from TSCreate()

Notes

TSPreStep() is typically used within time stepping implementations, so most users would not generally call this routine themselves.

Keywords

TS, timestep

See Also

TSSetPreStep(), TSPreStage(), TSPostStage(), TSPostStep()
