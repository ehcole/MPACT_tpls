TSGetStepRejections

 Gets the total number of rejected steps.

Synopsis

#include "petscts.h"  
PetscErrorCode TSGetStepRejections(TS ts,PetscInt *rejects)
Not Collective

Input Parameter

ts - TS context

Output Parameter

rejects - number of steps rejected

Notes

This counter is reset to zero for each successive call to TSSolve().

Keywords

TS, get, number
