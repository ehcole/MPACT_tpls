SNESMonitorResidual

 Monitors progress of the SNES solvers by calling VecView() for the residual at each iteration.

Synopsis

#include "petsc-private/snesimpl.h"   
PetscErrorCode  SNESMonitorResidual(SNES snes,PetscInt its,PetscReal fgnorm,void *dummy)
Collective on SNES

Input Parameters

snes - the SNES context
its - iteration number
fgnorm - 2-norm of residual
dummy - either a viewer or NULL

Keywords

SNES, nonlinear, vector, monitor, view

See Also

SNESMonitorSet(), SNESMonitorDefault(), VecView()
