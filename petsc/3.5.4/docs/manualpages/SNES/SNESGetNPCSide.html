SNESGetNPCSide

 Gets the preconditioning side.

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetNPCSide(SNES snes,PCSide *side)
Not Collective

Input Parameter

snes -iterative context obtained from SNESCreate()

Output Parameter

side -the preconditioning side, where side is one of 
      PC_LEFT - left preconditioning (default)
      PC_RIGHT - right preconditioning

Keywords

SNES, get, right, left, side, preconditioner, flag

See Also

SNESSetNPCSide(), KSPGetPCSide()
