SNESShellGetContext

 Returns the user-provided context associated with a shell SNES

Synopsis

#include "petscsnes.h"   
PetscErrorCode  SNESShellGetContext(SNES snes,void **ctx)
Not Collective

Input Parameter

snes -should have been created with SNESSetType(snes,SNESSHELL);

Output Parameter

ctx -the user provided context

Notes

This routine is intended for use within various shell routines

Keywords

SNES, shell, get, context

See Also

SNESCreateShell(), SNESShellSetContext()
