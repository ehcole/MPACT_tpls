SNESSetNPCSide

 Sets the preconditioning side.

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESSetNPCSide(SNES snes,PCSide side)
Logically Collective on SNES

Input Parameter

snes -iterative context obtained from SNESCreate()

Output Parameter

side -the preconditioning side, where side is one of 
      PC_LEFT - left preconditioning (default)
      PC_RIGHT - right preconditioning

Options Database Keys

-snes_pc_side <right,left> -

Keywords

SNES, set, right, left, side, preconditioner, flag
