SNESFASGetSmoother

 Gets the default smoother on a level.

Synopsis

#include "petscsnes.h"  
PetscErrorCode SNESFASGetSmoother(SNES snes, PetscInt level, SNES *smooth)

Input Parameters

snes - the multigrid context
level - the level (0 is coarsest) to supply

Output Parameters

smooth - the smoother
