DMDARestoreElements

 Returns an array containing the indices (in local coordinates) of all the local elements obtained with DMDAGetElements()

Synopsis

#include "petscdmda.h"   
PetscErrorCode  DMDARestoreElements(DM dm,PetscInt *nel,PetscInt *nen,const PetscInt *e[])
Not Collective

Input Parameter

dm - the DM object
nel - number of local elements
nen - number of element nodes
e - the local indices of the elements' vertices

See Also

DMDAElementType, DMDASetElementType(), DMDAGetElements()
