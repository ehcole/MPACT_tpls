DMGetPointSF

 Get the PetscSF encoding the parallel section point overlap for the DM.

Synopsis

#include "petscdm.h"     
PetscErrorCode DMGetPointSF(DM dm, PetscSF *sf)

Input Parameter

dm -The DM

Output Parameter

sf -The PetscSF

Note: This gets a borrowed reference, so the user should not destroy this PetscSF.

See Also

DMSetPointSF(), DMGetDefaultSF(), DMSetDefaultSF(), DMCreateDefaultSF()
