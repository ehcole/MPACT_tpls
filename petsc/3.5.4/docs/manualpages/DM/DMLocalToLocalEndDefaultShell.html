DMLocalToLocalEndDefaultShell

PetscErrorCode DMLocalToLocalEndDefaultShell(DM dm,Vec g,InsertMode mode,Vec l)

Input Arguments

dm - shell DM
g - the original local vector
mode - InsertMode
