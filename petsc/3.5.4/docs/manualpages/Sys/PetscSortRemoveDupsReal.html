PetscSortRemoveDupsReal

 Sorts an array of doubles in place in increasing order removes all duplicate entries

Synopsis

#include "petscsys.h"  
PetscErrorCode  PetscSortRemoveDupsReal(PetscInt *n,PetscReal v[])
Not Collective

Input Parameters

n - number of values
v - array of doubles

Output Parameter

n -number of non-redundant values
