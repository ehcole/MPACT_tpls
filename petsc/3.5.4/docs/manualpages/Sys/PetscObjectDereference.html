PetscObjectDereference

 Indicates to any PetscObject that it is being referenced by one less PetscObject. This decreases the reference count for that object by one.

Synopsis

#include "petscsys.h"    
PetscErrorCode  PetscObjectDereference(PetscObject obj)
Collective on PetscObject if reference reaches 0 otherwise Logically Collective

Input Parameter

obj -the PETSc object; this must be cast with (PetscObject), for example, PetscObjectDereference((PetscObject)mat);

Notes: PetscObjectDestroy(PetscObject *obj) sets the obj pointer to null after the call, this routine does not.

See Also

PetscObjectCompose(), PetscObjectReference()
