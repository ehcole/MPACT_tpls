KSPGetDiagonalScale

 Checks if KSP solver scales the matrix and right hand side

Synopsis

#include "petscksp.h" 
PetscErrorCode  KSPGetDiagonalScale(KSP ksp,PetscBool  *scale)
Not Collective

Input Parameter

ksp -the KSP context

Output Parameter

scale -PETSC_TRUE or PETSC_FALSE

Notes

BE CAREFUL with this routine: it actually scales the matrix and right hand side that define the system. After the system is solved the matrix and right hand side remain scaled unless you use KSPSetDiagonalScaleFix()

Keywords

KSP, set, options, prefix, database

See Also

KSPSetDiagonalScale(), KSPSetDiagonalScaleFix()
