MatAssembled

 Indicates if a matrix has been assembled and is ready for use; for example, in matrix-vector product.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatAssembled(Mat mat,PetscBool  *assembled)
Not Collective

Input Parameter

mat -the matrix

Output Parameter

assembled -PETSC_TRUE or PETSC_FALSE
