MatRARtNumeric

 Computes the matrix product C = R * A * R^T

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatRARtNumeric(Mat A,Mat R,Mat C)
Neighbor-wise Collective on Mat

Input Parameters

A - the matrix
R - the projection matrix

Output Parameters

C -the product matrix

Notes

C must have been created by calling MatRARtSymbolic and must be destroyed by the user using MatDeatroy().

This routine is currently only implemented for pairs of AIJ matrices and classes which inherit from AIJ. C will be of type MATAIJ.

See Also

MatRARt(), MatRARtSymbolic(), MatMatMultNumeric()
