PCGAMGInitializePackage

 This function initializes everything in the PCGAMG package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to PCCreate_GAMG() when using static libraries.

Synopsis

#include "petscpc.h" 
PetscErrorCode PCGAMGInitializePackage(void)
