TaoLineSearchComputeGradient

 Computes the gradient of the objective function

Synopsis

#include "petsctaolinesearch.h" 
PetscErrorCode TaoLineSearchComputeGradient(TaoLineSearch ls, Vec x, Vec g)
Collective on TaoLineSearch

Input Parameters

ls - the TaoLineSearch context
x - input vector

Output Parameter

g -gradient vector

Notes: TaoComputeGradient() is typically used within line searches so most users would not generally call this routine themselves.
