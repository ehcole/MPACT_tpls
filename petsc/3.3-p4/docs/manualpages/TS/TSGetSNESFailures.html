TSGetSNESFailures

 Gets the total number of failed SNES solves

Synopsis

#include "petscts.h"  
PetscErrorCode TSGetSNESFailures(TS ts,PetscInt *fails)
Not Collective

Input Parameter

ts -TS context

Output Parameter

fails -number of failed nonlinear solves

Notes

This counter is reset to zero for each successive call to TSSolve().

Keywords

TS, get, number
