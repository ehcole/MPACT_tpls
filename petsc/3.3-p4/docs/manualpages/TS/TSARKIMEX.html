TSARKIMEX

 ODE solver using Additive Runge-Kutta IMEX schemes These methods are intended for problems with well-separated time scales, especially when a slow scale is strongly nonlinear such that it is expensive to solve with a fully implicit method. The user should provide the stiff part of the equation using TSSetIFunction() and the non-stiff part with TSSetRHSFunction().

Notes

The default is TSARKIMEX2E, it can be changed with TSARKIMEXSetType() or -ts_arkimex_type

This method currently only works with ODE, for which the stiff part G(t,X,Xdot) has the form Xdot + Ghat(t,X).
