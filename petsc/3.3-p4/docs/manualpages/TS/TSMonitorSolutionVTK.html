TSMonitorSolutionVTK

 Monitors progress of the TS solvers by VecView() for the solution at each timestep.

Synopsis

#include "petscts.h"  
PetscErrorCode TSMonitorSolutionVTK(TS ts,PetscInt step,PetscReal ptime,Vec x,void *filenametemplate)
Collective on TS

Input Parameters

ts - the TS context
step - current time-step
ptime - current time
x - current state
filenametemplate - string containing a format specifier for the integer time step (e.g. %03D)

Notes

The VTK format does not allow writing multiple time steps in the same file, therefore a different file will be written for each time step. These are named according to the file name template.

This function is normally passed as an argument to TSMonitorSet() along with TSMonitorSolutionVTKDestroy().

Keywords

TS, vector, monitor, view

See Also

TSMonitorSet(), TSMonitorDefault(), VecView()
