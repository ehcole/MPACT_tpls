MatStencil

 Data structure (C struct) for storing information about a single row or column of a matrix as indexed on an associated grid.

Synopsis

typedef struct {
  PetscInt k,j,i,c;
} MatStencil;
Fortran usage is different, see MatSetValuesStencil() for details.

See Also
