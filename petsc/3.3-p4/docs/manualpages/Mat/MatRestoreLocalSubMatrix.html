MatRestoreLocalSubMatrix

 Restores a reference to a submatrix specified in local numbering

Not Collective

Input Arguments

mat - matrix to extract local submatrix from isrow - local row indices for submatrix iscol - local column indices for submatrix submat - the submatrix
