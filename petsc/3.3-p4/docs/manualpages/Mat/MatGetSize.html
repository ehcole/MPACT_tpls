MatGetSize

 Returns the numbers of rows and columns in a matrix.

Synopsis

#include "petscmat.h" 
PetscErrorCode  MatGetSize(Mat mat,PetscInt *m,PetscInt* n)
Not Collective

Input Parameter

mat -the matrix

Output Parameters

m - the number of global rows
n - the number of global columns

Note: both output parameters can be PETSC_NULL on input.

See Also

MatGetLocalSize()
