PetscFPTrapPush

Synopsis

#include "petscsys.h"  
PetscErrorCode PetscFPTrapPush(PetscFPTrap trap)
Not Collective

Input Arguments

trap -PETSC_FP_TRAP_ON or PETSC_FP_TRAP_OFF
