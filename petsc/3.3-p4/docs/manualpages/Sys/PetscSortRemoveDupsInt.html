PetscSortRemoveDupsInt

 Sorts an array of integers in place in increasing order removes all duplicate entries

Synopsis

#include "petscsys.h"  
PetscErrorCode  PetscSortRemoveDupsInt(PetscInt *n,PetscInt ii[])
Not Collective

Input Parameters

n - number of values
ii - array of integers

Output Parameter

n -number of non-redundant values
