PetscObjectGetClassName

 Gets the class name for any PetscObject

Synopsis

#include "petscsys.h"    
PetscErrorCode  PetscObjectGetClassName(PetscObject obj, const char *classname[])
Not Collective

Input Parameter

obj -any PETSc object, for example a Vec, Mat or KSP. Thus must be cast with a (PetscObject), for example, PetscObjectGetClassName((PetscObject)mat,&classname);

Output Parameter

classname -the class name
