PetscLikely

 hints the compiler that the given condition is usually TRUE

Synopsis

PetscBool  PetscUnlikely(PetscBool  cond)
Not Collective

Input Parameters

cond -condition or expression

Note: This returns the same truth value, it is only a hint to compilers that the resulting branch is likely.
