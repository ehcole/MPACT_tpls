AOInitializePackage

 This function initializes everything in the AO package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to AOCreate().

Synopsis

PetscErrorCode  AOInitializePackage(const char path[])

Input Parameter

path - The dynamic library path, or PETSC_NULL

Keywords

AO, initialize, package

See Also
