PCMGGetRestriction

 Gets the function to be used to restrict vector from level l to l-1.

Synopsis

#include "petscksp.h" 
#include "petscpcmg.h"   
PetscErrorCode  PCMGGetRestriction(PC pc,PetscInt l,Mat *mat)
Logically Collective on PC and Mat

Input Parameters

pc - the multigrid context
l - the level (0 is coarsest) to supply [Do not supply 0]

Output Parameter

mat -the restriction matrix

Keywords

MG, get, multigrid, restriction, level

See Also

PCMGGetInterpolation(), PCMGSetRestriction(), PCMGGetRScale()
