VecShift

 Shifts all of the components of a vector by computing x[i] = x[i] + shift.

Synopsis

#include "petscvec.h" 
PetscErrorCode  VecShift(Vec v,PetscScalar shift)
Logically Collective on Vec

Input Parameters

v - the vector
shift - the shift

Output Parameter

v -the shifted vector
