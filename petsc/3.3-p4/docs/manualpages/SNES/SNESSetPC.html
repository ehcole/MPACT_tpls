SNESSetPC

 Sets the nonlinear preconditioner to be used.

Synopsis

#include "petscsnes.h"  
#include "petscdmshell.h" 
#include "petscsys.h" 
PetscErrorCode SNESSetPC(SNES snes, SNES pc)
Collective on SNES

Input Parameters

snes - iterative context obtained from SNESCreate()
pc - the preconditioner object

Notes

Use SNESGetPC() to retrieve the preconditioner context (for example, to configure it using the API).

Keywords

SNES, set, precondition

See Also

SNESGetPC()
