DMCoarsen

 Coarsens a DM object

Synopsis

#include "petscdm.h"     
PetscErrorCode  DMCoarsen(DM dm, MPI_Comm comm, DM *dmc)
Collective on DM

Input Parameter

dm - the DM object
comm - the communicator to contain the new DM object (or MPI_COMM_NULL)

Output Parameter

dmc -the coarsened DM
