DMComplexSetConeSize

 Set the number of in-edges for this point in the Sieve DAG

Synopsis

#include "petscdmcomplex.h"   
PetscErrorCode DMComplexSetConeSize(DM dm, PetscInt p, PetscInt size)
Not collective

Input Parameters

mesh - The DMComplex
p - The Sieve point, which must lie in the chart set with DMComplexSetChart()
size - The cone size for point p

Output Parameter

Note

This should be called after DMComplexSetChart().

See Also

DMComplexCreate(), DMComplexGetConeSize(), DMComplexSetChart()
