DMComplexGetSupportSize

 Return the number of out-edges for this point in the Sieve DAG

Synopsis

#include "petscdmcomplex.h"   
PetscErrorCode DMComplexGetSupportSize(DM dm, PetscInt p, PetscInt *size)
Not collective

Input Parameters

mesh - The DMComplex
p - The Sieve point, which must lie in the chart set with DMComplexSetChart()

Output Parameter

size -The support size for point p

See Also

DMComplexCreate(), DMComplexSetConeSize(), DMComplexSetChart(), DMComplexGetConeSize()
