DMSetDefaultSection

 Set the PetscSection encoding the local data layout for the DM.

Synopsis

#include "petscdm.h"     
PetscErrorCode DMSetDefaultSection(DM dm, PetscSection section)

Input Parameters

dm - The DM
section - The PetscSection

Note: Any existing Section will be destroyed
