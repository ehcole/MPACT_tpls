DMMeshGetConeSize

 Return the number of in-edges for this point in the Sieve DAG

Synopsis

#include "petscdmmesh.h"   
PetscErrorCode DMMeshGetConeSize(DM dm, PetscInt p, PetscInt *size)
Not collective

Input Parameters

mesh - The DMMesh
p - The Sieve point, which must lie in the chart set with DMMeshSetChart()

Output Parameter

size -The cone size for point p

See Also

DMMeshCreate(), DMMeshSetConeSize(), DMMeshSetChart()
