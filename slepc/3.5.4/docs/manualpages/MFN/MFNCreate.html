MFNCreate

Creates the default MFN context.

Synopsis

#include "slepcmfn.h" 
PetscErrorCode MFNCreate(MPI_Comm comm,MFN *outmfn)
Collective on MPI_Comm

Input Parameter

comm - MPI communicator

Output Parameter

mfn - location to put the MFN context

Note

The default MFN type is MFNKRYLOV
