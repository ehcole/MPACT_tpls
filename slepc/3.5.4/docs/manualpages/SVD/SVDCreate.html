SVDCreate

Creates the default SVD context.

Synopsis

#include "slepcsvd.h" 
PetscErrorCode SVDCreate(MPI_Comm comm,SVD *outsvd)
Collective on MPI_Comm

Input Parameter

comm - MPI communicator

Output Parameter

svd - location to put the SVD context

Note

The default SVD type is SVDCROSS
