MFNReset

Resets the MFN context to the setupcalled=0 state and removes any allocated objects.

Synopsis

#include "slepcmfn.h" 
PetscErrorCode MFNReset(MFN mfn)
Collective on MFN

Input Parameter

mfn - matrix function context obtained from MFNCreate()
