RGGetOptionsPrefix

Gets the prefix used for searching for all RG options in the database.

Synopsis

#include "slepcrg.h" 
PetscErrorCode RGGetOptionsPrefix(RG rg,const char *prefix[])
Not Collective

Input Parameters

rg - the region context

Output Parameters

prefix - pointer to the prefix string used is returned

Notes: On the fortran side, the user should pass in a string 'prefix' of sufficient length to hold the prefix.

See Also

RGSetOptionsPrefix(), RGAppendOptionsPrefix()
