SVDReset

Resets the SVD context to the setupcalled=0 state and removes any allocated objects.

Synopsis

#include "slepcsvd.h" 
PetscErrorCode SVDReset(SVD svd)
Collective on SVD

Input Parameter

svd - singular value solver context obtained from SVDCreate()
